Demo AR desde QR (modelo 3D)

Escanea el QR que te dará la página del proyecto (o abre esta página desde el móvil). Si el navegador y el dispositivo lo permiten, pulsa el botón View in AR para colocar el modelo en el espacio real.

Nota: para que funcione en AR sin marcador necesitas que este archivo (y el .glb) se sirvan via HTTPS (GitHub Pages sirve así).

Modelo (ejemplo)

Instrucciones rápidas para usar tu propio .glb

  1. Descarga o crea un .glb de baja resolución y ponlo en la carpeta del proyecto.
  2. Sube todo a GitHub (o Netlify). Activa GitHub Pages desde Settings → Pages.
  3. Sustituye la URL del atributo src por https://TUUSUARIO.github.io/TU_REPO/tu_modelo.glb.
  4. Genera un QR que apunte a la URL pública de esta página (por ejemplo https://TUUSUARIO.github.io/TU_REPO/qr_version/) y pégalo en tu mapa.

Si tu móvil no ofrece el botón de AR, intenta abrir la página desde Chrome en Android (Scene Viewer) o Safari en iOS (Quick Look).
